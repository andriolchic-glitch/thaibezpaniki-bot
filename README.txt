=== THAILERT BOT — ИНСТРУКЦИЯ ===

ЧТО ДЕЛАЕТ ЭТОТ БОТ:
- Каждый час проверяет новости с тематических сайтов
- Фильтрует только то, что касается виз, въезда, документов для россиян
- Автоматически публикует в твой Telegram-канал

ФАЙЛЫ:
- bot.py — основной код бота
- requirements.txt — список библиотек
- README.txt — эта инструкция

КАК ЗАПУСТИТЬ — ШАГ ЗА ШАГОМ:

1. Открой файл bot.py в любом текстовом редакторе (Блокнот)
2. Найди строку: BOT_TOKEN = os.getenv("BOT_TOKEN", "ВСТАВЬ_СЮДА_ТОКЕН")
3. Замени ВСТАВЬ_СЮДА_ТОКЕН на свой токен от @BotFather
4. Найди строку: CHANNEL_ID = "@thailert"
5. Замени @thailert на username твоего канала

ДАЛЕЕ — ЗАГРУЗКА НА СЕРВЕР (Railway.app):
1. Зайди на https://railway.app
2. Зарегистрируйся через GitHub (github.com)
3. Нажми "New Project" → "Deploy from GitHub"
4. Загрузи эти файлы
5. В настройках добавь переменную: BOT_TOKEN = твой_токен

ВАЖНО:
Добавь бота в свой канал как АДМИНИСТРАТОРА,
иначе он не сможет публиковать сообщения!
